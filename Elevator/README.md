## Projects

### Elevator

A terminal-based elevator simulator that accepts commands like `up`, `down`, `ground`, and `exit` to control the elevator while keeping track of its current floor.

**Concepts practiced:** objects, functions, `while` loops, `switch` statements, conditional statements, terminal input using `readline-sync`, state management

**To run:**

```bash
cd elevator
node elevator.js
```

---

## Setup

This repo uses `readline-sync` for terminal-based user input.

Install dependencies once at the root:

```bash
npm install
```
